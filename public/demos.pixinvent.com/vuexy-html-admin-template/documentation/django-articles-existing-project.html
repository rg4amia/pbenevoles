Existing Project Migration

Here, we answer your most common question, how to use our template in your existing project?

Info: Our template is based on Bootstrap 5. We assume that your existing project would be on Bootstrap 5. Otherwise, it may break the structure and UI of the template.

Vuexy Django Admin Template

The Vuexy admin template is not just a theme. It's more than it. It is a fully functional template that has much-interconnected functionality.

It's not like a WordPress theme to install in an existing project. We always suggest you, move your existing project into our Vuexy Django admin template.

If you try to move our template into your existing project it may break the UI and some functionality. It is not impossible to achieve the same approach. You have to deeply understand, the structure and every functionality of our template to achieve that.

Basic Steps to migrate an existing project into Vuexy
  1. First, you need add your app name to setting.py and urls.py.
  2. Now, extend our layouts as per your requirement in your existing pages. We have also provided snippets django-template
    3. {% extends layout_path %}

{% load i18n %}

{% block title %}//Your Page title{% endblock %}

{% block vendor_css %}
{{ block.super }}
// your vendor style files goes here
{% endblock vendor_css %}

{% block vendor_js %}
{{ block.super }}
// your vendor script files goes here
{% endblock vendor_js %}

{% block page_css %}
{{ block.super }}
// your page specific style files goes here
{% endblock page_css %}

{% block page_js %}
{{ block.super }}
// your page specific script files goes here
{% endblock page_js %}

{% block content %}
// your page content goes here
{% endblock %}
  3. Move your assets like SCSS, JS files into the resource folder. If you use our existing folder to place your assets then you no need to define your compilation in the webpack.mix.js file.
  4. Add your project dependency into requirements.txt or package.json file.
  5. Add your existing pages name into menus JSON files to add your page's links in the sidebar menu. You can find our menus files below location.
    6. templates/layout/partials/menu/horizontal/horizontalMenu.json
templates/layout/partials/menu/vertical/verticalMenu.json
  6. Also, add your page's name into locale/django.po files to get the translation.
    7. 📂 locale
├── 📂 en
│   └──📂 LC_MESSAGES
│       └──📄 django.po
├── 📂 fr
|   └──📂 LC_MESSAGES
|       └──📄 django.po
├── 📂 en
│   └──📂 LC_MESSAGES
│       └──📄 django.po
└── 📂 fr
    └──📂 LC_MESSAGES
        └──📄 django.po
    Important: After adding the name, which will be translated, remember to python manage.py compilemessages.
  7. To search your pages from navbar search, add them into the below-mentioned search JSON files.
    8. /src/assets/json/search-vertical.json
/src/assets/json/search-horizontal.json
  8. Above are some basic steps to migrate your project into Vuexy.👍
